# Les structure du DOM

jQuery permet de :

* copier un élément
  * clone()
* insérer  un élément du DOM
  * wrap(), append, prepend(), html(), after(), before(), etc.
* ou supprimer un élément
  * detach(), unwrap()
* accéder aux attributs personnalisés des éléments, de type `data-*`
  * replaceWith()
